Hardware Guide

Core Components

This project has been tested with the hardware listed below. While it should work with more powerful Raspberry Pi models, these are the recommended components for the most cost-effective setup.

  • Raspberry Pi Zero 2W

    Board with headers

    ~€20

  • MicroSD Card

    16GB or 32GB recommended (tested), but 8GB should work

    €4-15

  • Official Power Supply (optional)

    If not using an existing charging cable

    €8

  • USB-C to micro-USB adapter

    For connecting to modern computers and to avoid having to buy a new micro-USB cable or the official power supply

    ~€1
Subtotal: ~€33 to €43

Optional Extensions

Battery Power

PiSugar 3 or similar battery pack for portable use

Custom Case

3D printed case or use the one included with some display models. A vertical orientation case would be ideal for this application.
